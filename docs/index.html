本地运行指南

需要 Python 3.10+ 与 Pygame。可选地放入音效文件提升沉浸感。

1. 安装依赖

pip install -r requirements.txt

2. 启动游戏

python main.py

3. 自定义音效

将音效放入 assets/sounds/
